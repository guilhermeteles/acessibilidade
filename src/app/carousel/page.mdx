# Carousel

## Critérios WCAG e eMAG
<Properties fullWidth>
  <Property name="WCAG 2.1.1 / eMAG Diretriz 5" type="Controles acessíveis e rótulos">
    O Carousel deve ser navegável via teclado, com controles de navegação claramente rotulados (por exemplo, `aria-label="Próximo slide"`).
  </Property>
  <Property name="WCAG 2.2.2 / eMAG Diretriz 9" type="Evitar animações automáticas">
    Evite a rotação automática dos slides, ou forneça um controle para pausar ou parar a animação.
  </Property>
  <Property name="WCAG 2.4.7 / eMAG Diretriz 5" type="Estado de foco visível">
    O foco deve ser visível nos controles do Carousel.
  </Property>
</Properties>
