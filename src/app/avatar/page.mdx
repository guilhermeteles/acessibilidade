# Avatar

## Critérios WCAG e eMAG
<Properties fullWidth>
  <Property name="WCAG 1.1.1 / eMAG Diretriz 3" type="Texto alternativo">
    O Avatar deve ter texto alternativo descritivo `alt` caso seja mais que decorativo (por exemplo, o nome do usuário). Se for puramente decorativo, use `aria-hidden="true"`. 
  </Property>
  <Property name="WCAG 1.4.3 / eMAG Diretriz 7" type="Contraste de cores">
    Se o Avatar incluir texto, o contraste entre o texto e o fundo deve atender aos requisitos de contraste.
  </Property>
</Properties>
