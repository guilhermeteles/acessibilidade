# Avatar

## Critérios WCAG e eMAG
<Properties fullWidth>
  <Property name="WCAG 1.1.1 / eMAG Diretriz 3" type="Texto alternativo">
    O Avatar deve ter texto alternativo descritivo `alt` caso seja mais que decorativo (por exemplo, o nome do usuário). Se for puramente decorativo, use `aria-hidden="true"`. 
  </Property>
  <Property name="WCAG 1.4.3 / eMAG Diretriz 7" type="Contraste de cores">
    Se o Avatar incluir texto, o contraste entre o texto e o fundo deve atender aos requisitos de contraste.
  </Property>
</Properties>


## Desenvolvimento
<Row>
  <Col>
    - Implementar `aria-hidden` para avatares decorativos ou `alt` para descritivos.
  </Col>
  <Col>
  ```bash {{ title: 'Acessibilidade no Desenvolvimento - Avatar Icônico' }}
      <br-avatar 
        name="Jhon Doe"
        density="small"
        aria-hidden="true">
      </br-avatar>
    ```
    ```bash {{ title: 'Acessibilidade no Desenvolvimento - Avatar Fotográfico' }}
      <br-avatar
        name="Jhon Doe"
        image="https://picsum.photos/id/823/400"
        alt="Avatar de Jhon Doe">
      </br-avatar>
    ```
    ```bash {{ title: 'Acessibilidade no Desenvolvimento - Avatar com Tooltip' }}
      <br-avatar
        name="Jhon Doe"
        tooltip-text="Avatar de Jhon Doe"
        tooltip-place="right"
        alt="Avatar de Jhon Doe">
      </br-avatar>
    ```
  </Col>
