# Loading

## Critérios WCAG e eMAG
<Properties fullWidth>
  <Property name="WCAG 4.1.2 / eMAG Diretriz 10" type="Informação para usuários de tecnologias assistivas">
    Durante o carregamento, deve haver um anúncio de que o conteúdo está carregando e quando o carregamento for concluído. Utilize `aria-live="polite"` ou `aria-busy="true"` para notificar usuários de leitores de tela que o conteúdo está sendo carregado e quando estiver pronto.
  </Property>
  <Property name="WCAG 2.3.1 / eMAG Diretriz 9" type="Evitar loops de animação automática ou piscantes">
   Certifique-se de que a animação de carregamento não cause desconforto (ex.: não piscar ou se repetir muito rapidamente). Evite animações que possam causar convulsões ou distração excessiva.
  </Property>
</Properties>
