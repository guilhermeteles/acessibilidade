# Icon

## Critérios WCAG e eMAG
<Properties fullWidth>
  <Property name="WCAG 1.1.1 / eMAG Diretriz 3" type="Alternativas de texto">
    Os ícones devem ter uma alternativa de texto apropriada para que leitores de tela possam descrever sua função. Isso pode ser feito com `aria-label` ou `aria-hidden="true"` se o ícone for puramente decorativo e não tiver uma função interativa.
  </Property>
  <Property name="WCAG 1.4.3 / eMAG Diretriz 7" type="Contraste de cores">
    O ícone deve ter contraste suficiente em relação ao fundo para ser legível por todos os usuários, especialmente para pessoas com baixa visão.
  </Property>
  <Property name="WCAG 2.4.7 / eMAG Diretriz 5" type="Foco visível em ícones interativos">
    Se o ícone for clicável ou interativo, deve haver um foco visual claro quando o ícone for selecionado ou ativado pelo teclado.
  </Property>
</Properties>
