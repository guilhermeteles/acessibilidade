# Menu

## Critérios WCAG e eMAG
<Properties fullWidth>
  <Property name="WCAG 1.1.1 / eMAG Diretriz 3" type="Navegabilidade por teclado">
    Certifique-se de que todos os itens do menu sejam navegáveis via teclado. Utilize as teclas de seta para navegar entre os itens do menu e `Enter` para selecionar um item.
  </Property>
  <Property name="WCAG 2.4.7 / eMAG Diretriz 5" type="Foco visível">
    Cada item do menu deve ter um foco visível quando selecionado via teclado, indicando claramente qual opção está ativa.
  </Property>
  <Property name="WCAG 1.4.3 / eMAG Diretriz 7" type="Contraste adequado">
    O texto do menu deve ter contraste suficiente em relação ao fundo para garantir que todos os itens sejam legíveis. O contraste mínimo deve ser de `4.5:1`.
  </Property>
  <Property name="WCAG 3.3.2 / eMAG Diretriz 6" type="Rotulagem e estrutura">
    Garanta que o menu tenha uma estrutura clara e rótulos acessíveis para que os usuários compreendam as opções.
  </Property>
</Properties>
