# Divider

## Critérios WCAG e eMAG
<Properties fullWidth>
  <Property name="WCAG 1.4.1 / eMAG Diretriz 3" type="Uso sem dependência exclusiva de cores">
    O divisor deve ser usado como um elemento de separação visual, mas sem depender exclusivamente da cor para transmitir sua função. Se o divisor for importante para a estrutura ou navegação, pode ser útil incluir um rótulo ou descrição adicional.
  </Property>
  <Property name="WCAG 1.4.3 / eMAG Diretriz 7" type="Contraste de cores">
    O divisor deve ter contraste adequado com o fundo para garantir que seja visível, especialmente para usuários com baixa visão. Certifique-se de que o divisor se destaca, mas não prejudique a legibilidade de outros elementos ao redor.
  </Property>
</Properties>
