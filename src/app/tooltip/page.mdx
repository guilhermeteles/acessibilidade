# Tooltip

## Critérios WCAG e eMAG
<Properties fullWidth>
  <Property name="WCAG 2.1.1 / eMAG Diretriz 5" type="Exibição com teclado">
    O tooltip deve ser acessível via teclado, ou seja, deve aparecer quando o elemento é focado (não apenas no `hover` do mouse). Deve ser possível exibir a dica de ferramenta com teclas como `Tab` ou `Enter`.
  </Property>
  <Property name="WCAG 1.1.1 / eMAG Diretriz 3" type="Texto alternativo descritivo">
    O conteúdo do tooltip deve ser acessível para leitores de tela. Utilize `aria-describedby` para garantir que o tooltip seja lido junto com o elemento que ele descreve.
  </Property>
  <Property name="WCAG 2.2.1 / eMAG Diretriz 4" type="Tempo de exibição">
   Certifique-se de que o tooltip permaneça visível tempo suficiente para que o usuário possa lê-lo, ou que ele possa ser controlado (por exemplo, manter visível enquanto o foco está no elemento).
  </Property>
  <Property name="WCAG 2.4.7 / eMAG Diretriz 5" type="Contraste adequado">
    O texto dentro do tooltip deve ter contraste adequado em relação ao fundo.
  </Property>
</Properties>
