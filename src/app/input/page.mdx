# Input

## Critérios WCAG e eMAG
<Properties fullWidth>
  <Property name="WCAG 1.1.1 / 3.3.2 / eMAG Diretriz 6" type="Rótulos visíveis e instruções">
    Todos os campos de entrada devem ter rótulos claros e, se necessário, instruções adicionais para o usuário.
  </Property>
  <Property name="WCAG 2.1.1 / eMAG Diretriz 5" type="Navegabilidade por teclado">
    O campo de entrada deve ser acessível via teclado e ter um estado de foco visível.
  </Property>
  <Property name="WCAG 3.3.3 / eMAG Diretriz 6" type="Erro e validação">
    Caso ocorra um erro ao preencher o campo, deve ser fornecida uma mensagem clara, com sugestões de correção.
  </Property>
</Properties>
