# Button

## Critérios WCAG e eMAG
<Properties fullWidth>
  <Property name="WCAG 1.1.1 / eMAG Diretriz 3" type="Rótulos acessíveis para botões">
    Todo botão deve ter um rótulo claro, visível ou através de `aria-label`, que descreva sua função.
  </Property>
  <Property name="WCAG 2.1.1 / eMAG Diretriz 5" type="Navegabilidade por teclado">
    O botão deve ser operável usando o teclado, com foco claro e visível para os usuários que navegam sem mouse.
  </Property>
  <Property name="WCAG 1.4.3 / eMAG Diretriz 7" type="Contraste de cores">
    O botão deve ter contraste adequado entre o texto e o fundo para garantir legibilidade. Verifique a relação de contraste mínima de `4.5:1`.
  </Property>
  <Property name="WCAG 2.4.7 / eMAG Diretriz 5" type="Feedback visual">
    Deve haver uma clara distinção visual quando o botão está focado ou ativo.
  </Property>
</Properties>
