# Table

## Critérios WCAG e eMAG
<Properties fullWidth>
  <Property name="WCAG 1.1.1 / eMAG Diretriz 3" type="Cabeçalhos de tabela descritivos">
    Certifique-se de que cada célula de cabeçalho `<th> `esteja devidamente marcada e associada às células de dados correspondentes `<td>` usando `scope`, `headers` ou `aria-labelledby`. Isso permite que leitores de tela associem corretamente os dados aos cabeçalhos.
  </Property>
  <Property name="WCAG 2.1.1 / eMAG Diretriz 5" type="Navegabilidade por teclado">
    A tabela deve ser navegável por teclado. Isso significa que os usuários devem ser capazes de navegar célula por célula e também saltar entre seções da tabela.
  </Property>
  <Property name="WCAG 2.4.7 / eMAG Diretriz 5" type="Contraste de cores">
    O texto da tabela, incluindo linhas de cabeçalho e dados, deve ter contraste suficiente em relação ao fundo.
  </Property>
   <Property name="WCAG 2.4.7 / eMAG Diretriz 5" type="Mensagens de erro">
    Caso haja formulários ou inputs dentro da tabela, forneça mensagens de erro claras para entradas inválidas.
  </Property>
</Properties>
