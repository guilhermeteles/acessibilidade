# Textarea

## Critérios WCAG e eMAG
<Properties fullWidth>
  <Property name="WCAG 1.3.1 / 3.3.2 / eMAG Diretriz 6" type="Rótulos e instruções claras">
    Cada área de texto deve ter um rótulo claro, visível e associado corretamente ao campo usando `label for`. Se necessário, forneça instruções sobre o que deve ser inserido na área de texto.
  </Property>
  <Property name="WCAG 2.1.1 / eMAG Diretriz 5" type="Navegabilidade por teclado">
    A área de texto deve ser totalmente navegável via teclado, com foco visível e claro quando o usuário interagir com o campo.
  </Property>
  <Property name="WCAG 3.3.1 / eMAG Diretriz 6" type="Mensagens de erro">
    Caso uma entrada inválida seja detectada, forneça mensagens de erro claras, indicando o problema e como corrigi-lo.
  </Property>
  <Property name="WCAG 1.4.4 / eMAG Diretriz 7" type="Redimensionamento acessível">
    Garanta que a área de texto possa ser redimensionada sem perder a funcionalidade ou a visibilidade do conteúdo.
  </Property>
</Properties>
