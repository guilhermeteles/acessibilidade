# Breadcrumb

## Critérios WCAG e eMAG
<Properties fullWidth>
  <Property name="WCAG 2.1.1 / eMAG Diretriz 5" type="Navegabilidade por teclado">
    O Breadcrumb deve ser completamente navegável via teclado, com cada link representando claramente sua posição na trilha.
  </Property>
  <Property name="WCAG 2.4.7 / eMAG Diretriz 5" type="Estado de foco visível">
    Quando os itens do Breadcrumb são focados, deve haver um indicador visual claro que o link está ativo (ex.: mudança de cor, borda, etc.).
  </Property>
  <Property name="WCAG 2.4.4 / eMAG Diretriz 6" type="Texto descritivo">
    Os links devem ter textos claros e descritivos, explicando claramente para onde levam. Se o último item não for clicável (indica a página atual), ele deve ser devidamente marcado como tal (com `aria-current="page"`).
  </Property>
  <Property name="WCAG 1.4.3 / eMAG Diretriz 7" type="Contraste de cores">
    Certifique-se de que o texto do Breadcrumb tenha contraste adequado com o fundo (mínimo `4.5:1`) para garantir sua legibilidade.
  </Property>
</Properties>
